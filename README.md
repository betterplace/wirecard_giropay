# Wirecard Giropay

An implementation of the wirecard giropay interface for giropay payments.


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'wirecard_giropay'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wirecard_giropay
